# High Frequency Highway - Landing Page & Funnel

## Project Overview
E-commerce landing page and sales funnel for High Frequency Highway. Built with React + TypeScript + Vite + Tailwind CSS + Shopify integration.

## Mandatory Skill Routing Rules

All work on this project MUST be filtered through the appropriate skill before implementation. Never make design or copy decisions without consulting the relevant skill first.

### Visual Design Decisions → UI/UX Pro Max Skill
**Trigger:** Any work involving layout, colors, typography, spacing, components, animations, responsive design, or visual styling.

Before making ANY visual decision:
1. Read `.claude/skills/ui-ux-pro-max/SKILL.md` and search the relevant data files in `.claude/skills/ui-ux-pro-max/data/`
2. Use the skill's style, color, typography, and UX guideline databases to inform choices
3. Follow the skill's priority hierarchy: Accessibility → Touch/Interaction → Performance → Layout → Typography/Color → Animation → Style

This applies to: page layouts, component design, color palettes, font selections, button styles, hover states, shadows, gradients, spacing, responsive breakpoints, dark mode, animations, and all visual elements.

### Copywriting & Sales Copy → Master Copywriting & Persuasion Skill
**Trigger:** Any work involving headlines, subheadlines, CTAs, product descriptions, body copy, email copy, or any user-facing text.

Before writing ANY copy:
1. Invoke the `anthropic-skills:master-copywriting-persuasion-skill` to apply proven frameworks (Sugarman, Brunson, Kennedy, Vitale)
2. Use persuasion frameworks for headlines, hooks, and calls-to-action
3. Apply sales psychology principles to product descriptions and value propositions

This applies to: hero headlines, subheadlines, CTA buttons, product descriptions, feature descriptions, testimonial framing, urgency/scarcity messaging, and all customer-facing text.

### E-commerce & Sales Funnel Strategy → NEPQ Sales + High-Ticket Course Builder Skills
**Trigger:** Any work involving funnel structure, product positioning, offer stacking, pricing, upsells, cart flow, or conversion optimization.

Before making ANY funnel or sales strategy decision:
1. Invoke `anthropic-skills:nepq-sales-coach` for prospect qualification and objection handling
2. Invoke `anthropic-skills:high-ticket-course-builder` for offer structuring, value stacking, and funnel architecture
3. Apply Grand Slam Offer principles to pricing and packaging

This applies to: funnel page sequence, offer structure, pricing display, upsell/cross-sell flows, cart abandonment strategy, urgency elements, guarantee framing, and conversion optimization.

## Skill Routing Summary

| Decision Type | Skill to Use | Key Focus |
|---|---|---|
| Visual/UI/UX | UI/UX Pro Max (`.claude/skills/ui-ux-pro-max/`) | Style, color, typography, layout, accessibility |
| Copy/Text | `master-copywriting-persuasion-skill` | Headlines, CTAs, persuasion frameworks |
| Funnel/Sales | `nepq-sales-coach` + `high-ticket-course-builder` | Offer structure, conversion, objection handling |
| Brand Voice | `nlp-representational-systems` | Sensory language, communication style mirroring |

## Enforcement
- Do NOT skip skill consultation — even for "small" text or style changes
- When skills conflict, prioritize: Accessibility > Conversion > Aesthetics
- Always cite which skill/framework informed a decision when presenting options to the user
